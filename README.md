Rock, Paper, Scissors!
----------------------

Play a delightful game of **rock**, **paper**, **scissors**!

Currently the game can only be played as **Human Vs. Human**.

Rules
-----

- Rock beats scissors.
- Paper beats Rock.
- Scissors beats paper.

How to run tests
-----------------

- Currently tests are being run via IntelliJ.


System Requirements
-------------------

- Java version "9.0.1"
